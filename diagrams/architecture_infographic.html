LinkedIn Thought Leadership Agent

Solution Architecture & Workflow Reference
9 SKILLS MCP ENABLED PATH A: SKILLS PACK
Problem: Senior leaders have deep expertise but lack time to consistently post thought leadership on LinkedIn. Solution: 9-skill agentic pipeline that interviews, plans, drafts, refines, benchmarks, archives, and publishes a 6-week content series. Platform: Claude Code / Manus AI
🧠
Skill 0: Master Orchestrator
Sequences all skills, maintains state (Strategic Intent + Roadmap + Drafts), enforces HITL gates, handles error recovery
Pipeline Phases
1
Intent Discovery
Skill 1
5-Question Strategic Interview:
  • Q1 Target Audience
  • Q2 Core Message
  • Q3 Personal Anecdote
  • Q4 Call to Action
  • Q5 Tone & Style
Gate: All 5 Qs answered before proceeding
IN: Topic title
OUT: Strategic Intent Document
2
Content Strategist
Skill 2
6-Week Narrative Arc:
  • Wk1: The Hook (Pain Point)
  • Wk2: The Framework
  • Wk3: The Story (Anecdote)
  • Wk4: The Tactics
  • Wk5: The Vision
  • Wk6: The Call (CTA)
Each week builds on the previous
IN: Strategic Intent
OUT: 6-Week Roadmap
3
Draft & Refine
Skills 3-5
Three-stage creation:
  • Skill 3 Draft Architect
    150-300 word drafts, anti-AI-ism rules
  • Skill 4 Voice Refiner
    Ghostwriter: removes AI patterns, adds authority
  • Skill 5 Engagement Optimizer
    LinkedIn hooks, formatting, hashtags, visuals
IN: Intent + Roadmap
OUT: 6 Optimized Posts
✋
HITL Checkpoints
5a + 6a
Checkpoint 5a — User Feedback:
  • Present all 6 posts for review
  • Collect: Approve / Revise / Reject
  • Loop to Skills 3-5 if revisions needed
Checkpoint 6a — Benchmark:
  • Skill 6 scores 5 metrics (1-5)
  • Present scorecard to user
  • Flag any score < 3.0
Both gates require explicit user approval
IN: Optimized Posts
OUT: Approved Series + Scores
7
Archive Manager
Skill 7
  • Packages full session to Markdown
  • Intent + Roadmap + Posts + Scores
  • Saves to archive/ folder
8
LiGo MCP Publisher
Skill 8
  • Publishes via LinkedIn MCP
  • Schedule or post immediately
  • Verify & return live URL
Optional — requires MCP server

Benchmark Metrics (Skill 6)

Actionability (1-5) • Voice Consistency (1-5) • Strategic Depth (1-5) • Narrative Cohesion (1-5) • LinkedIn Optimization (1-5)

Key Design Decisions

Sequential pipeline with 2 HITL gates ensures quality without full automation risk. Skills are modular: swap Skill 4 for a different voice engine without changing the pipeline.
